# Приклад передачі між сесіями

## Виконано

- Додано підтримку імпорту Markdown
- Додано базовий список документів у renderer

## Зламано або не верифіковано

- Імпорт успішний для `.md`, але не вдається для великих файлів `.txt`
- Застосунок запускається, але детальний вигляд ще не підключено

## Наступний найкращий крок

- Виправити шлях імпорту `.txt`
- Верифікувати імпорт наскрізно
- Потім додати панель деталей документа
