# Приклад рубрики оцінювача

Використовуйте оцінювання від 1 до 5 для кожного виміру:

- Заземленість: чи пов'язані відповіді чітко з імпортованими джерелами?
- Якість цитування: чи є посилання на джерела видимими та конкретними?
- Функціональність: чи може користувач завершити потік запитань і відповідей?
- Узгодженість продукту: чи відчувається робочий процес інтегрованим?
