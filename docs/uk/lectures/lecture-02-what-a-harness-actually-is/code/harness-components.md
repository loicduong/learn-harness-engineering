# Приклад компонентів harness

Для агента написання коду, що працює в локальному репозиторії:

- Модель:
  сам LLM

- Harness:
  - системний промпт
  - AGENTS.md
  - bash-інструмент
  - інструменти читання/запису файлів
  - доступ до git
  - локальна файлова система
  - стартові скрипти
  - команди для запуску тестів
  - stop hooks
  - перевірки лінтером
  - цикл оцінювача

Якщо змінити будь-яку з наведених частин harness, зміниться й ефективний агент.
