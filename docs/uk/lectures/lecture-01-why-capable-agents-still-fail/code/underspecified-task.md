# Приклад недостатньо специфікованої задачі

Збудуй десктопний застосунок для бази знань із відповідями на запитання за допомогою AI.

Обмеження:

- Не вказані
- Команда запуску не надана
- Структура тек не визначена
- Модель даних не описана
- Явних критеріїв завершення немає

Типові результати такого промпту:

- агент вигадує структуру ad hoc
- застосунок може компілюватися, але запускатися нестабільно
- UI може з'явитися раніше, ніж з'явиться будь-який придатний для використання шлях інтеграції/запиту даних
- агент нерідко зупиняється після косметичного успіху
