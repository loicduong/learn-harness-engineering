# Шаблон циклу мети

> Напишіть вашу мету як документ ось так, потім передайте його `/goal` або агенту maker.
> Чим конкретніше і перевіреніше, тим вища якість циклу.

## Мета

<!-- Одне речення, що описує, що робити -->

Реалізувати функцію XX з повним покриттям модульними тестами.

## Критерії прийняття

<!-- Умови завершення, що перевіряються машиною — кожну можна перевірити командою -->

- [ ] `npm test` проходить повністю
- [ ] Звіт про покриття показує покриття модуля XX ≥ 80%
- [ ] `npm run lint` має нульових помилок
- [ ] Перевірка типів TypeScript проходить (`npx tsc --noEmit`)
- [ ] Увесь новий код відповідає стандартам кодування проекту

## Обсяг

<!-- Чітко вкажіть, чого НЕ чіпати -->

### Можна чіпати

- Усі файли в `src/xx/`
- Тестові файли в `tests/`
- Пов'язані файли визначення типів

### Не чіпати

- Вхідний файл `src/main.ts`
- Міграції схеми бази даних
- Версії залежностей у `package.json` (крім випадків явної необхідності)
- Файли конфігурації CI/CD

## Метод перевірки

<!-- Як перевіряти після кожного кроку -->

1. Після реалізації запустіть ці команди по порядку:
   1. `npx tsc --noEmit` — перевірка типів
   2. `npm run lint` — стиль коду
   3. `npm test` — модульні тести
   4. `npx vitest --coverage` — перевірка покриття

2. Виправте будь-які невдачі перед переходом до наступного кроку.
3. Коли все проходить, запустіть повний набір e2e-тестів (якщо доступний).

## Умови зупинки

- Усі критерії прийняття проходять ✅
- Досягнуто максимальну кількість ходів: 20
- Немає прогресу протягом 3 послідовних раундів (одна й та ж помилка з'являється знову і знову)
- Блокуюча проблема, яку не можна вирішити самостійно (наприклад, відсутня залежність, проблеми з середовищем)

## Як працювати

1. Спочатку прочитайте `AGENTS.md` та `feature_list.json`, щоб зрозуміти структуру проекту та наявні функції.
2. Напишіть дизайн-підхід перед тим, як чіпати код.
3. Перевіряйте після завершення кожного підзавдання.
4. Якщо застрягли більше ніж на 2 раунди, змініть підхід або спростіть.
5. Оновлюйте прогрес наприкінці кожного раунду.
