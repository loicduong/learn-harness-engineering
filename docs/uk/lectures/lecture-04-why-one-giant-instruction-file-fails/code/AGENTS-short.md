# AGENTS.md

## Починайте звідси

- Прочитайте `docs/ARCHITECTURE.md`
- Прочитайте `docs/PRODUCT.md`
- Використовуйте `npm run dev` для запуску застосунку
- Використовуйте `npm run check` перед тим, як позначити роботу завершеною

## Жорсткі правила

- Не змінюйте межі між Electron main/preload/renderer без попереднього читання
  `docs/ARCHITECTURE.md`
- Не позначайте функцію завершеною без верифікації
- Залишайте чистий стан для наступної сесії
