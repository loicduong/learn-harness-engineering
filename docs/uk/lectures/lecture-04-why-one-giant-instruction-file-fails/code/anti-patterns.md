# Антипатерни файлів інструкцій

- Розміщення всіх знань про репозиторій в одному файлі
- Повторення одного й того самого правила в кількох місцях
- Збереження застарілих правил, які ніхто не перевіряє
- Написання умовних інструкцій, настільки специфічних, що вони рідко застосовуються
- Вбудовування об'ємних посібників з інструментів у стартовий контекст
