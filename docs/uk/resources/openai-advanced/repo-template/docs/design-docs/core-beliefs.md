# Ключові принципи

- Репозиторій є системою запису для агентів.
- `AGENTS.md` — це маршрутизатор, а не енциклопедія.
- Докази верифікації важливіші за впевненість.
- Одне обмежене завдання краще, ніж багато напівзавершених.
- Повторний зворотний зв'язок від людей має ставати повторно використовуваними правилами harness.
- Очищення та спрощення є частиною випуску, а не доопрацюванням після нього.
- Якщо агент не може знайти факт у репозиторії, вважайте цей факт операційно недоступним.
