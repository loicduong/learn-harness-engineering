# Індекс документів проєктування

- `core-beliefs.md`: принципи роботи агентів та тривалі норми проєкту

## Запропоновані

## Застарілі

## Правила підтримки

- Кожен документ проєктування повинен мати власника або тригер для оновлення.
- Видаляйте застарілі документи або позначайте їх як застарілі замість того, щоб залишати їх дрейфувати.
- Пов'язуйте активні плани виконання з документами проєктування, від яких вони залежать.
