# Схема бази даних

Використовуйте цю папку для згенерованих або похідних артефактів, які агенти мають
мати змогу перевіряти без зворотного інжинірингу з коду.

## Джерело

- Згенеровано з: `[command or source path]`
- Останнє оновлення: `YYYY-MM-DD`

## Примітки

- Не редагуйте вручну згенеровані розділи.
- Перегенеруйте цей файл, коли змінюється базова схема.
