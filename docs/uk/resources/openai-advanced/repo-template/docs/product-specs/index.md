# Індекс специфікацій продукту

Використовуйте цю папку для актуальних специфікацій поведінки продукту, видимої користувачеві.

## Активні специфікації

- `new-user-onboarding.md`

## Правила

- Специфікації мають описувати поведінку, видиму користувачеві, та критерії прийнятності.
- Якщо реалізація розходиться зі специфікацією, оновіть один із них в тій самій сесії.
- Тримайте цей індекс актуальним, щоб новий агент міг швидко дізнатися про обсяг продукту.
