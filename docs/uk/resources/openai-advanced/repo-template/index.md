# Розширений шаблон репозиторію

Скопіюйте цей стартовий набір у реальний репозиторій, якщо хочете отримати
документаційну поверхню у стилі OpenAI, орієнтовану на агентів, замість лише мінімального harness.

## Порядок копіювання

1. Скопіюйте `AGENTS.md` і `ARCHITECTURE.md` до кореня репозиторію.
2. Скопіюйте всю директорію `docs/`.
3. Заповніть `docs/PRODUCT_SENSE.md`, `docs/QUALITY_SCORE.md` і
   `docs/RELIABILITY.md` насамперед.
4. Додайте свій перший активний план у `docs/exec-plans/active/`.
5. Тримайте вхідні файли короткими та направляйте деталі до пов'язаних документів.

## Для чого оптимізовано цей шаблон

- тривалий локальний контекст репозиторію
- прогресивне розкриття інформації замість одного великого файлу інструкцій
- явний життєвий цикл планів
- відстеження якості з часом
- читабельні межі для агентів і людей

Ставтеся до кожного файлу тут як до стартового. Замінюйте заповнювачі, приклади та
зразки команд реальними деталями вашого проєкту перед використанням.
