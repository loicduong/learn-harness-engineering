# Контрольний список чистого стану

- [ ] Стандартний шлях запуску досі працює.
- [ ] Стандартний шлях верифікації досі виконується.
- [ ] Поточний прогрес зафіксований у журналі прогресу.
- [ ] Стан функцій відображає те, що дійсно проходить верифікацію, а не лише заявлено.
- [ ] Жодний незавершений крок не залишений без документування.
- [ ] Наступна сесія може продовжити роботу без ручного виправлення.
