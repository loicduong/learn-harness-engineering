# Рубрика оцінювача

Використовуйте цю рубрику після реалізації та перед фінальним прийняттям.

| Категорія | Питання | Оцінка (0-2) | Примітки |
| --- | --- | --- | --- |
| Коректність | Чи відповідає реалізована поведінка запитаній функції? |  |  |
| Верифікація | Чи були дійсно виконані необхідні перевірки з доказами? |  |  |
| Дисципліна обсягу | Чи залишалася сесія в межах обраної функції? |  |  |
| Надійність | Чи витримує результат перезапуск або повторний запуск без виправлень? |  |  |
| Супроводжуваність | Чи є код і документація достатньо зрозумілими для наступної сесії? |  |  |
| Готовність до передачі | Чи може нова сесія продовжити роботу, спираючись лише на артефакти репозиторію? |  |  |
