# Бібліотека ресурсів

Ця папка перетворює методи курсу на готові до копіювання шаблони та компактні
довідники, які можна використовувати в реальному репозиторії.

## Коли використовувати

Починайте тут, якщо ви хочете, щоб Codex, Claude Code або інший агент кодування
працював впродовж кількох сесій без постійного повторного виведення налаштувань,
статусу та обсягу задач.

Особливо корисно, коли:

- робота охоплює кілька сесій
- функцій багато і їх легко залишити напівзробленими
- агенти схильні оголошувати перемогу завчасно
- кроки запуску доводиться відкривати заново щоразу

## Починайте тут

Для мінімального налаштування почніть із:

- кореневих інструкцій: [`templates/AGENTS.md`](./templates/AGENTS.md) або [`templates/CLAUDE.md`](./templates/CLAUDE.md)
- стану функцій: [`templates/feature_list.json`](./templates/feature_list.json)
- журналу прогресу: [`templates/claude-progress.md`](./templates/claude-progress.md)
- довідника по сценарію завантаження: `docs/uk/resources/templates/init.sh`

Потім додайте:

- передачу сесії: [`templates/session-handoff.md`](./templates/session-handoff.md)
- контрольний список чистого виходу: [`templates/clean-state-checklist.md`](./templates/clean-state-checklist.md)
- рубрику оцінювача: [`templates/evaluator-rubric.md`](./templates/evaluator-rubric.md)

Якщо вам потрібна повна структура репозиторію у стилі OpenAI з публікації
"Harness engineering", скористайтесь розширеним пакетом:

- [`openai-advanced/index.md`](./openai-advanced/index.md)

## Структура бібліотеки

- [`templates/`](./templates/index.md): шаблони для копіювання в реальний репозиторій
- [`reference/`](./reference/index.md): нотатки щодо методів, процес запуску та карти режимів збою
- [`openai-advanced/`](./openai-advanced/index.md): розширений скелет репозиторію,
  документи системи обліку та шаблони управління для агентів

## Рекомендований мінімальний пакет

- `AGENTS.md` або `CLAUDE.md`
- `feature_list.json`
- `claude-progress.md`
- `init.sh`

Цих чотирьох файлів достатньо, щоб зробити більшість робочих процесів з агентами
помітно стабільнішими.

Коли репозиторій переростає в довгострокову систему з кількома доменами, активними
планами, оцінюванням якості та політиками надійності, переходьте до пакету
[`openai-advanced/`](./openai-advanced/index.md) замість того, щоб надто
розтягувати мінімальний пакет.
