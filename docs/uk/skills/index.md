# Skills

Цей каталог містить вбудовані Skills для AI-агентів, що входять до цього курсу. Skills — це самодостатні шаблони промптів, які можуть завантажувати AI-агенти з кодування (Claude Code, Codex, Cursor, Windsurf тощо) для виконання спеціалізованих завдань.

## harness-creator

Продакшн-рівневий skill з Harness Engineering для AI-агентів кодування. Він допомагає створювати, оцінювати та покращувати п'ять основних підсистем harness: інструкції, стан, верифікацію, область застосування та життєвий цикл сесії.

### Що він робить

- **Створення harness з нуля** — AGENTS.md, списки функцій, робочі процеси верифікації
- **Покращення наявних harness** — оцінювання за п'ятьма підсистемами з пріоритизованими покращеннями
- **Проєктування безперервності сесій** — збереження пам'яті, відстеження прогресу, процедури передачі стану
- **Застосування продакшн-патернів** — пам'ять, інжиніринг контексту, безпека інструментів, координація між агентами

### Швидкий старт

Файли Skills розміщені в репозиторії за адресою [`skills/harness-creator/`](https://github.com/walkinglabs/learn-harness-engineering/tree/main/skills/harness-creator).

```bash
npx skills add walkinglabs/learn-harness-engineering --skill harness-creator
```

Щоб використовувати зі Claude Code, скопіюйте каталог `harness-creator/` до шляху Skills вашого проєкту або вкажіть агенту на файл SKILL.md.

### Довідкові патерни

Skill включає 7 сфокусованих довідкових документів:

| Патерн | Коли використовувати |
|--------|----------------------|
| Memory Persistence | Агент забуває між сесіями |
| Skill Runtime | Пакування повторно використовуваних робочих процесів як Skills |
| Context Engineering | Управління бюджетом контексту, JIT-завантаження |
| Tool Registry | Безпека інструментів, контроль паралелізму |
| Multi-Agent Coordination | Паралелізм, спеціалізовані робочі процеси |
| Lifecycle & Bootstrap | Хуки, фонові завдання, ініціалізація |
| Gotchas | 15 неочевидних режимів відмов із виправленнями |

### Шаблони

Skill містить готові до використання шаблони:

- `agents.md` — каркас AGENTS.md із робочими правилами
- `feature-list.json` — JSON Schema + приклад списку функцій
- `init.sh` — стандартний скрипт ініціалізації
- `progress.md` — шаблон журналу прогресу сесії
- `session-handoff.md` — шаблон передачі стану сесії

### Скрипти

Skill також включає прості Node.js-скрипти для генерації каркасів, валідації, HTML-звітів оцінювання та структурних звітів бенчмарків.

### Як було створено цей Skill

`harness-creator` розроблено з використанням методології **skill-creator** — офіційного мета-skill Anthropic для створення, тестування та ітерації Skills для агентів. skill-creator надає структурований робочий процес (чернетка → тестування → оцінювання → ітерація) із вбудованими запускальниками eval, оцінювачами та переглядачем бенчмарків.
